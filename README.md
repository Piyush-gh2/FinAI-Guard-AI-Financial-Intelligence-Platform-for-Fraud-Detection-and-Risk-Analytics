# FinAI-Guard: AI Financial Intelligence Platform

FinAI-Guard is an AI-powered financial analytics platform designed to detect fraudulent banking transactions and generate automated investigation reports using machine learning and generative AI.

This project demonstrates how artificial intelligence can improve financial security, risk detection, and transaction monitoring in modern banking systems.

---

## Project Features

- Fraud detection using Machine Learning (XGBoost)
- Financial risk analysis for transactions
- Automated investigation report generation using Generative AI (GPT)
- Interactive dashboard using Streamlit
- Data preprocessing and feature engineering
- Model training and evaluation

---
